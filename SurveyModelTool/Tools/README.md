# SurveyModelTool

This is the README for the SurveyModelTool package.

## Description
SurveyModelTool is a Python package designed to process and analyze travel diary survey data. It provides tools for reading, preprocessing, and visualizing survey data.

## Installation
To install the package in editable mode, use the following command:

```bash
pip install -e .
```
